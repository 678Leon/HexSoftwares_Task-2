 Project 1: Advanced Web Scraper
Description
A multi-page web scraper built with Python to extract book data from `books.toscrape.com`.
It scrapes 5 pages automatically and saves structured data into a CSV file.


 Features
- Multi-page scraping (5 pages)
- Extracts: Book Title, Price, Stock, Star Rating, Book URL
- Auto-saves to CSV (structured format)
- Error handling & server-friendly delay
- Clean, professional output

Libraries Used
- requests
- beautifulsoup4
- lxml
